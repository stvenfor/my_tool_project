# 第 3 周：无字幕短场 + 整段理解（Day 15–21）

**焦点**：正常语速精听；每天至少 1 场「无字幕 → 英文字幕核对」。  
**策略**：抓重读实词、人名、否定、情态；见 [phrases.md](phrases.md) 第 3 周锚点。  
**语速**：默认 1.0×；卡句用 0.75× 拆开后再合回 1.0×。

每日约 **2.5–3h**：精听 50 + 影子 20 + 电影 60 + 短语复习 10。

---
