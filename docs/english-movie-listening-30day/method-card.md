# 每日方法卡（可贴墙 / 手机备忘）

## 硬规则

1. 精听每段 **≤ 1–2 分钟**，短段重复 > 整片精听
2. 字幕：Day 1–10 中英 → Day 11–20 仅英文 → Day 21–30 先无字幕再英文字幕核对
3. 每天生词/短语 **≤ 15**，优先电影口语块，不背词典表
4. 每天必须有 **影子跟读或朗读台词**（有输出）
5. 同片/同剧重复，不要天天换片

## 精听 3 遍流程

| 遍数 | 做什么 |
|------|--------|
| 第 1 遍 | 抓大意：谁？在哪？发生什么？ |
| 第 2 遍 | 写关键词（名词/动词/否定词），不求完整句子 |
| 第 3 遍 | 逐句听；仍不懂再 0.75×，然后回到 1.0× |

然后：对稿标漏听/连读 → 影子跟读（滞后约 0.5 秒）→ 记短语卡片。

## 听懂策略（第 3 周起强化）

优先抓：**重读实词 → 人名 → 否定（not/never/can't）→ 情态（should/would/might）→ 谁对谁做什么**。  
不要死磕每个冠词 the/a。

## 时间表模板（2.5–3h）

| 时段 | 内容 |
|------|------|
| 早 30–40 min | 语音或热身跟读 + 1 段精听 |
| 碎片 20 min | 只听不看昨天材料 |
| 晚 90–120 min | 精听收尾 + 影子跟读 + 电影泛听 + 词汇 |

睡前不学新内容；可被动再听今天跟读音频一遍。
