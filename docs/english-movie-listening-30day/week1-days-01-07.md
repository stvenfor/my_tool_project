# 第 1 周：听清「声音」（Day 1–7）

**焦点**：音素、弱读、慢速精听。电影只做 3–5 分钟短场 + 中英字幕。  
**参照**：[method-card.md](method-card.md) · [phrases.md](phrases.md) 第 1 周表

每日合计约 **2.5h**：语音 15 + 慢速精听 40 + 短语 20 + 电影 40–50 + 复盘 10。

---

## Day 1 — gonna / wanna / gotta

**语音（15）**
- [ ] Rachel's English 或同类：gonna / wanna / gotta
- [ ] 跟读 10 句含这三者的例句

**慢速精听（40）**
- [ ] 材料：VOA Learning English（任选 1 篇，切 **≤2 min**）
- [ ] 3 遍精听 → 对稿 → 跟读

**短语（20）**
- [ ] 从 phrases 第 1 周表练 #1–5 + I'm gonna / I wanna / Gotta

**电影（40–50）**
- [ ] *Pursuit of Happyness* 或 Friends S01E01：**同一场 3–5 min**
- [ ] 中英字幕；跟读 5–8 句含 gonna/wanna 的对白

**复盘（10）**
- [ ] 写下今天听漏最多的 3 个音变：
  1. ________  2. ________  3. ________

## Day 2 — 弱读 a / the / of / to / for

**语音（15）**
- [ ] 弱读：a→ə, the→ðə, of→əv/ə, to→tə, for→fər

**慢速精听（40）**
- [ ] 新 VOA/ESL 段 ≤2 min；完整 3 遍流程
- [ ] 对稿时特别圈出弱读漏听

**短语（20）**

**电影（40–50）**
- [ ] **同一部**续看下一场 3–5 min（或 Friends 同集下一场）
- [ ] 中英字幕；暂停跟读 5 句

**复盘（10）**
- [ ] 三音变：________ / ________ / ________
