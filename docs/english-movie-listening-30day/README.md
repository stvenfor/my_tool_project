# 30 天四六级词句（10× 扩容）

入口：[`daily-vocab-30day.md`](daily-vocab-30day.md)

## 每天份量

| 项目 | 数量 |
|------|------|
| 单词（含国际音标） | **80** |
| 短语（含音标） | **40** |
| 不同类型句子 | **30** |
| 词根词缀 | 每组主题若干 |

合计约 **2400 词 + 1200 短语 + 900 句型例句**。

## 学法

每天约 **90–120 分钟**（可拆早 40 词 + 晚 40 词）：音标跟读 → 英译中 → 中译英 → 短语整句；睡前复习昨天约 20 词。
