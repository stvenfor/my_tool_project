# Infographic Engine 提示词骨架（CFFEX 日报美化）

复制后填入当日 JSON 数值。Image 1 = 风格参考，Image 2 = 数据底图。

```
模板：Infographic Engine / 信息图引擎（case 334、case 1、case 8）

任务：中文金融数据日报竖版信息图（抖音 9:16），商务 PPT 精修风。

参考图角色：
- 图1：唯一版式/视觉风格参考（白底、细线分区、深蓝标题编号、红=加多、青绿=加空、扁平图表）
- 图2：唯一数据与文案来源；日期、中文、数值、柱向、署名必须以图2为准

构图（自上而下）：
1. 页眉：左「中信期货 · 净持仓日报」；大号日期「{{date_label}}」；副标题「来源于网络 · 机构持仓数据」；右上圆形头像 +「{{logo_handle}}」
2. 今日要点浅红横幅：今日要点 · {{peak_symbol}} {{加多|加空}} {{peak_abs}} 手为最大边际变化
3. 01 各品种净持仓：IH / IF / IC / IM 四卡（符号与加多/加空徽章）
4. 02 持仓变化对比：正负柱图，平方根刻度，图例加多/加空，数值标签不重叠
5. 03 市场概览：中信整体 {{citic_total}}；前20机构净空单（存量）缩写万手 + 精确手；今日净买入 {{net_buy_total}}
6. footer：数据来源于网络，仅供参考 · 投资有风险，入市需谨慎

视觉：白底、克制阴影、对齐网格、充足留白；禁止花哨装饰、乱码、错数、文字重叠。
比例：9:16 竖版高清 PNG。
```

`peak_*`：取 `citic_by_symbol` 中绝对值最大的品种。
