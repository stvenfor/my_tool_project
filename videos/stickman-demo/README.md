# 换我 · Gray Stickman

灰色火柴人活在别人的画布里。趁创作者去倒咖啡，他爬出草稿纸，钻进隔壁文件夹——那里躺着一只只画了左翼的蝴蝶角色。桌上只留下两个字：**换我。**

## 分镜（56s）

| 时间 | 场景 |
|------|------|
| 0–5s | 片头「换我」 |
| 4.5–16s | 死亡循环：挨打 / 坠落 / 炸飞 |
| 15.5–22s | 创作者去倒咖啡 |
| 21.5–30s | 爬出纸边 → 钻进文件夹 |
| 29.5–36s | 遇见半成品左翼角色 |
| 35.5–42s | 桌上纸条「换我。」 |
| 41.5–49s | 第二天：灰色，有左翼 |
| 48.5–56s | 笔尖悬空，迟迟不落 |
